# 访问修饰符

## 静态属性
```js
class Computer{
    // 定义一个静态属性 username
    static get username(){
        return "java"
    }
}
console.log(Computer.username)  // java
const lx = new Computer()
console.log(lx.username)  // undefined
```
## 静态方法
```js
class Computer{
    constructor(){}
    static show(){
        console.log('show...')
    }
}
console.log(new Computer().show)  // undefined
Computer.show()  // show...
```
